Responsive Web Design (RWD 響應式網頁設計)

  • 1. RWD，讓內容如有水般地適應各種形狀的容器，能在每個裝置尺寸正常瀏覽。
  • 2. CSS 的媒體查詢(Media Queries)負責響應是最主要的工作
  • 3. Media Query的breakpoint(斷點)為不同裝置尺寸設定通過的定位點，當通過設定的位置時就套用該尺寸的樣式。
  • 4. 行動裝置優先的漸進增強概念，基於小尺寸開始思考內容的設計模式。
  • 5. 使用相對安為設定寬、高或大小(rem, %, vh, vw)，字體大小使用 rem，容器寬度使用 %
